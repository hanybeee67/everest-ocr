## 1. 보안 아키텍처 개요 (Security Architecture)

본 시스템은 **2중 보안 계층**을 적용하여 관리자 권한과 회원 개인정보를 강력하게 보호하고 있습니다.

| 보호 대상 | 적용 기술 | 설명 |
| :--- | :--- | :--- |
| **관리자 비밀번호** | **SHA-256 Hashing** | 단방향 암호화 기술로, 해커가 DB를 탈취해도 원래 비밀번호를 복원할 수 없습니다. (256비트 해시) |
| **회원 개인정보** | **Fernet Encryption** | **AES-128 (CBC)** 암호화와 **HMAC-SHA256** 무결성 인증을 결합한 산업 표준 암호화 방식입니다. (256비트 키 사용) |
